COLOR SCHEME
------------

Background Colors:
- Window:     #121212 (RGB 18, 18, 18)   - Very dark grey
- Panels:     #19191C (RGB 25, 25, 28)   - Dark grey
- Cards:      #202024 (RGB 32, 32, 36)   - Medium dark grey
- Hover:      #3C3C41 (RGB 60, 60, 65)   - Lighter grey

Accent Colors:
- Epic Blue:  #0079D6 (RGB 0, 121, 214)  - Epic Games blue
- Text:       #E6E6E6 (RGB 230, 230, 230) - Light grey
- Disabled:   #808080 (RGB 128, 128, 128) - Grey

UI ELEMENTS
-----------

Buttons:
- Normal:   Dark background (#2D2D30), light text
- Hover:    Slightly lighter background (#3C3C41)
- Active:   Epic Blue background (#0079D6), white text
- Disabled: Grey background, dimmed text

Text Inputs:
- Background: Dark (#2D2D30)
- Text:       Light (#E6E6E6)
- Border:     Subtle grey
- Focus:      Epic Blue border

Cards:
- Background: #202024
- Border Radius: 4px
- Shadow: Subtle dark shadow
- Hover: Slightly lighter

Game Cards Layout:
- Image area: 230x130 pixels (placeholder with centered text)
- Title: Below image, 14pt bold
- Buttons: Bottom of card, 14pt


INTERACTIONS
------------

1. Search Box:
   - Real-time filtering as you type
   - Searches game titles
   - Case-insensitive

2. Filter Buttons:
   - "All Games": Shows entire library
   - "Installed": Shows only installed games
   - Active filter highlighted in Epic Blue

3. Game Cards:
   - Hover effect: Card slightly lighter
   - Click Install: Button shows "Installing...", becomes disabled
   - After install: Shows "Play" and "Uninstall" buttons
   - Click Play: Launches game process
   - Click Uninstall: Removes game, shows "Install" again

4. Status Bar:
   - Shows current operation status
   - "Clear" button to dismiss
   - Auto-updates during operations


RESPONSIVE BEHAVIOR
-------------------

Window Width: 1200px (default)
- Shows 4-5 cards per row

Window Width: 800px (minimum)
- Shows 2-3 cards per row

Window can be resized, cards automatically reflow to fit available space.
